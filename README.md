# Quiz Application

An interactive quiz application developed using HTML, CSS, and JavaScript.
The project focuses on implementing logic-based question handling and
user interaction through a simple and clean interface.

## Features
- Multiple-choice questions
- Instant feedback on answers
- Score calculation
- Interactive user interface

## Technologies Used
- HTML
- CSS
- JavaScript

## Concepts Covered
- Conditional statements
- JavaScript arrays and objects
- DOM manipulation
- User input handling

## Purpose
This project was built to practice JavaScript logic, control flow,
and front-end development skills while creating an interactive application.
